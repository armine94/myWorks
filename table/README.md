## CLIENT REQUIREMENTS
1) Node v12.18 >=
2) Npm v6 >=
3) Chrome 89
4) React >= 16.8.0


## **Client side requirements**
### Install Node Js
		sudo apt update
		sudo apt install nodejs


### Install node package manager
		sudo apt install npm


### Install client dependencies
		open a new terminal
		cd table
		npm ci


### Start client
		npm start
